# Git 기초 학습 (2026-04-23)

## 오늘 배운 것
- 오늘 공부의 흐름 : main 최신화 → 새 브랜치 → 제출 문서 작성/저장 → add → commit → push → PR
- 순서 :
- 1. git pull origin main -> main 최신화
- 2. git branch [브랜치 명] -> 작업 공간 만들기
- 3. 만들고 나서 add, commint, push, PR (New Pull Requests)
