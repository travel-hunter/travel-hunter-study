# 팀원별 개인 학습 기록

  각 팀원이 자신의 학습 내용을 자유롭게 기록하는 공간입니다.

  ## 사용 방법
  1. 본인 이름(또는 GitHub 아이디)으로 하위 폴더 생성
  2. 학습 내용을 `YYYY-MM-DD-주제.md` 형식으로 작성
  3. 커밋 후 푸시

  ## 폴더 예시
```
  members/
  ├── kimjihoon/
  │   ├── 2026-04-23-git-basics.md
  │   └── 2026-04-24-linux-commands.md
  └── leechulsoo/
      └── 2026-04-23-git-basics.md
```
